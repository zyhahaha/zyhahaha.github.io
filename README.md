# zyhahaha.github.io
Blog

### Shell 命令
``` bash
$ hexo new postName   # 新建文章
$ hexo new page postName  # 新建页面
$ hexo generate -d  # 生成网站并部署到GitHub上
$ hexo server   # 开启本地服务器预览
```

### 环境依赖
Nodejs v12.0
